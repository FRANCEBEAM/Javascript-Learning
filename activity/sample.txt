The most common types of nodes in XML are − Document Node − Complete XML document structure is a document node. Element Node − Every XML element is an element node. This is also the only type of node that can have attributes. Attribute Node − Each attribute is considered an attribute node. It contains information about an element node, but is not actually considered to be children of the element. Text Node − The document texts are considered as text node. It can consist of more information or just white space.
